# Cyberbullying Detection System

A machine learning-based cyberbullying detection system developed to identify harmful and abusive text content on social media platforms.

---

## 📌 Overview
This project uses Natural Language Processing (NLP) and machine learning algorithms to detect cyberbullying-related text automatically. The system helps classify messages as bullying or non-bullying based on textual analysis.

---

## 🚀 Technologies Used
- Python
- Machine Learning
- Natural Language Processing (NLP)
- Naive Bayes
- Random Forest
- Scikit-learn
- Pandas
- NumPy

---

## ✨ Features
- Text preprocessing and cleaning
- Cyberbullying text classification
- Machine learning model training
- Dataset analysis and prediction
- Improved detection accuracy using multiple classifiers

---

## 📊 Algorithms Used
- Naive Bayes Classifier
- Random Forest Classifier

---

## 🎯 Objective
The main objective of this project is to identify and reduce cyberbullying content on online platforms using machine learning techniques.
